# Context

## Glossary

- Language variant URL: A sitemap URL whose first path segment is a language code, such as `/de/...` or `/zh-TW/...`.
- Canonical URL: The single URL we keep for a page when multiple language variant URLs point to the same page identity.
- Page identity: The comparison key built from scheme, host, query, and path after removing a leading language code from the first path segment.
